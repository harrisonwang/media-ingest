# 可接受使用政策

本政策用于约束 Mingest 的使用方式，避免被用于违法或高风险场景。

## 你可以做什么

- 以个人归档/备份为目的，在法律与平台规则允许范围内下载你有权保存的内容
- 对你拥有版权或已获授权的内容进行转码、剪辑、封装等本地处理

## 你不可以做什么

- 侵犯版权或其他合法权益（例如下载、复制、传播你无权保存的内容）
- 将下载内容用于公开传播、分发、售卖或搬运（除非你拥有权利或授权）
- 绕过 DRM、加密、付费墙、访问控制或其他技术保护措施
- 大规模抓取/爬取/批量搬运内容，或将工具用于自动化滥用
- 以服务形式对外提供“在线解析/代下/代保存”
- 诱导他人提供账号密码或 cookies，或共享/售卖/公开他人的账户登录信息

## 执行

若发现明显滥用行为，我们可能在文档与产品层面添加限制（例如更严格的提示、默认参数调整等）。开源项目的 Issue/讨论区也可能对相关内容进行删除或锁定。
