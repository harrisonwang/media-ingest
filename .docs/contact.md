# 联系与投诉指引

## 反馈与问题

- 功能建议 / Bug 反馈：优先使用 GitHub Issues
- 安全问题：请不要公开披露到 Issue；建议私下联系维护者

## 权利人投诉

Mingest 不托管任何内容，也不提供在线解析/代下服务。若你认为某个使用场景可能侵犯了你的合法权益，建议提供以下信息，便于我们判断是否需要在文档或产品层面增加更明确的边界与提示：

- 你的身份与权利证明（如适用）
- 涉及的平台/链接类型（不建议直接提供个人隐私信息）
- 你认为构成侵权或违规的原因
- 你希望我们采取的措施（例如完善文档边界、补充提示、移除示例等）

说明：我们无法控制第三方用户如何使用本地工具，也无法代替平台进行内容治理。
